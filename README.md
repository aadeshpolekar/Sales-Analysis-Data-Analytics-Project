# Sales Analysis Data Analytics Project

## Overview
This is an end-to-end data analytics project where sales data is analyzed using SQL, Python, and Excel.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Microsoft Excel

## Project Steps
1. Data Cleaning using Pandas and NumPy
2. Data Analysis using SQL Queries
3. Data Visualization using Matplotlib
4. Dashboard Creation using Excel (Pivot Tables, Charts, Slicers)

## Key Insights
- Identified top-performing product categories
- Analyzed sales trends and patterns
- Compared performance across categories

## Dashboard
Interactive Excel dashboard with slicers and charts for dynamic filtering.
